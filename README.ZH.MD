# Processing 版球球大作战

## 描述

- 由 `Processing 3.5.4` 实现球球大作战的一个简单实例

- 操作： 上下左右+空格

![404 Not Found](https://hrlu.cn:8799/share/assets/balls.png)

## 运行

1. 下载 [`Processing 3.5.4`](https://processing.org/download/)

2. 安装模块 `Minim 2.2.2`

   - Java -> 添加模式

   - Libraries -> 搜索 Minim

   - Install

3. 或下载 [release]() 版本
